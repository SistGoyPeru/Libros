# Prefacio

### ¿Por qué este libro?

JavaScript es el lenguaje más popular de la web. Está en todos lados: en los botones que presionas, en los formularios que llenas, en las animaciones que te sorprenden, en las aplicaciones que usas a diario. Pero aprenderlo puede ser abrumador si empiezas con manuales técnicos llenos de sintaxis abstracta.

Este libro es diferente.

**JavaScript Básico: El Poder de la Interactividad** es una novela educativa. Acompaña a Camila Ríos, una diseñadora UX que nunca ha programado, en su misión de convertir una plataforma educativa estática en una experiencia interactiva. Ella comete tus mismos errores, hace tus mismas preguntas, y celebra tus mismos triunfos.

### ¿Cómo usar este libro?

Cada capítulo técnico tiene tres partes:

1. **Narrativa** — La historia de Camila y su equipo en "AprendeYa". Un problema del mundo real que necesita una solución JavaScript.
2. **Explicación técnica** — Los conceptos que Camila aprende para resolver sus problemas. Con ejemplos claros y funcionales.
3. **Enigmas** — Ejercicios para que practiques. Al final del libro encontrarás las soluciones.

### ¿Qué necesitas?

- Un navegador web moderno (Chrome, Firefox, Edge)
- Visual Studio Code (o cualquier editor de código)
- Curiosidad y ganas de crear

### Un viaje interactivo

Al terminar este libro, no solo sabrás qué pasó con la plataforma "AprendeYa". Habrás aprendido JavaScript desde las variables hasta la manipulación del DOM, pasando por funciones, arrays, objetos, eventos, formularios, almacenamiento local y consumo de APIs.

Y descubrirás que JavaScript no es solo un lenguaje de programación. Es el poder de hacer que la web cobre vida.
