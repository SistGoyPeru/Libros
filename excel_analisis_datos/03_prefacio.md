# Prefacio

## ¿Por qué otro libro de Excel?

Porque la mayoría de libros de Excel enseñan funciones. Este libro enseña **a pensar con datos**.

El análisis de datos no es una lista de fórmulas. Es un proceso: preguntar, obtener, limpiar, explorar, modelar, comunicar. Excel es la herramienta, pero el método es lo que importa.

### Cómo usar este libro

Cada capítulo presenta un problema real de análisis de datos, resuelto paso a paso con Excel. Al final, un enigma práctico para que apliques lo aprendido.

Los archivos de ejemplo están en `codigos/`.

### Requisitos

- Excel 2019 o Microsoft 365
- Conocimientos básicos de Excel (saber qué es una celda y una fórmula)
- Ganas de pensar como analista de datos
