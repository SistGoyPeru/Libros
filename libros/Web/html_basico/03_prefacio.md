# Prefacio

### ¿Por qué este libro?

Has visto el título. "HTML Básico: El Código de la Web". Suena a manual, a curso técnico, a aburrimiento. Y sin embargo, este es un libro para aprender HTML de una manera completamente diferente.

Este libro nació de una convicción: **la mejor manera de aprender una herramienta es necesitarla de verdad.**

No te voy a engañar con ejercicios de "crea un párrafo con color rojo". Aquí vas a ayudar a Valentina Rivas a terminar el sitio web del periódico "La Voz del Barrio", a publicar artículos, a organizar contenido, a crear formularios, y a descubrir que detrás de cada etiqueta HTML hay una historia que merece ser contada.

### ¿Cómo usar este libro?

Cada capítulo tiene tres partes:

1. **Narrativa** — La historia avanza. Valentina descubre el legado de Don Eduardo, enfrenta desafíos, comete errores, aprende y crece.
2. **Explicación técnica** — Los conceptos de HTML que Valentina usa para resolver sus problemas. Explicados con claridad, con ejemplos prácticos.
3. **Enigmas** — Ejercicios para que tú mismo practiques lo aprendido. Al final del libro encontrarás las soluciones.

### ¿Qué necesitas?

- Un editor de texto (recomiendo Visual Studio Code, que es gratuito)
- Un navegador web (Chrome, Firefox, Edge o Safari)
- Curiosidad y ganas de aprender

### Un viaje, no un destino

Al terminar este libro, no solo sabrás cómo Valentina salvó "La Voz del Barrio". También habrás aprendido HTML desde la estructura básica hasta formularios, multimedia, SEO y accesibilidad web.

Y quizás, solo quizás, descubras que HTML no es solo un lenguaje de marcado. Es el código que da forma a la web, el lienzo donde pintar tus ideas, la puerta que abre tu contenido al mundo.
