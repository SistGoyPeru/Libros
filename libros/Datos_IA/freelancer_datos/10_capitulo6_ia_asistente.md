# Capítulo 6: IA como Asistente de Datos

## 6.1 La IA Cambió las Reglas del Juego

Si no usas IA en tu flujo de trabajo como freelancer de datos, estás compitiendo con una mano atada atrás. No se trata de que la IA haga tu trabajo — se trata de que hagas **más** en **menos tiempo**.

## 6.2 Herramientas de IA Recomendadas

Las herramientas de IA recomendadas incluyen ChatGPT para análisis, código, explicaciones y copywriting; Claude para análisis largos con gran contexto de 100k tokens; GitHub Copilot para autocompletado de código en VS Code; Midjourney o DALL-E para gráficos conceptuales en informes; y Perplexity para búsqueda rápida con IA.

## 6.3 Prompts que Funcionan para Analistas de Datos

### 6.3.1 Análisis de datos

```
Prompt: "Tengo un CSV con columnas [fecha, producto, ventas, región].
Necesito un análisis de tendencias de ventas por región.
¿Qué aproximación me recomiendas y qué visualizaciones usarías?"

Prompt: "Actúa como un data analyst senior. Revisa este SQL:
[PEGA TU SQL]. Optimízalo para rendimiento y legibilidad.
Explícame qué cambios hiciste y por qué."
```

### 6.3.2 Creación de informes

```
Prompt: "Necesito escribir un informe ejecutivo sobre la caída de ventas
del último trimestre. Estos son los datos clave: [DATOS].
Escribe un resumen ejecutivo de 3 párrafos: situación, análisis,
recomendaciones. Tono profesional pero no técnico."
```

### 6.3.3 Explicaciones para el cliente

```
Prompt: "Explica qué es un JOIN en SQL a un gerente de marketing
que no sabe programar. Usa una analogía del mundo real.
Máximo 3 párrafos. Sin jerga técnica."
```

### 6.3.4 Automatización de tareas repetitivas

```
Prompt: "Escribe un script en Python que lea todos los archivos CSV
de una carpeta, los combine en un solo DataFrame, limpie los nulos
y exporte a Parquet. Añade logging y manejo de errores."
```

## 6.4 Flujo de Trabajo con IA

### 6.4.1 Mi flujo diario como freelancer

El flujo diario recomendado con IA incluye: por la mañana, revisar emails y pendientes con ayuda de Claude para priorizar; a media mañana, codificar con Copilot y ChatGPT para depuración; por la tarde, analizar datos con ChatGPT mientras el freelancer interpreta; y al final del día, redactar informes con IA para luego pulirlos y personalizarlos.

### 6.4.2 Lo que la IA NO puede hacer (aún)

- Entender el contexto real de tu cliente
- Construir relaciones de confianza
- Tomar decisiones estratégicas con matices
- Reemplazar tu criterio profesional

**La IA es tu asistente, no tu reemplazo.** El valor que vendes como freelancer es tu criterio, tu experiencia y tu capacidad de conectar datos con decisiones de negocio.

## 6.5 Ética y Privacidad con IA

Las reglas de privacidad con IA incluyen: nunca subir datos sensibles de clientes a ChatGPT o Claude sin usar versiones enterprise o locales; anonimizar siempre los datos antes de compartirlos con herramientas cloud; usar modelos locales como Ollama o Llama 3 para datos confidenciales; informar a los clientes si se usa IA en sus proyectos; y verificar siempre los outputs porque los LLMs pueden alucinar.
