# Capítulo 8: Recursos Web y Plataformas Recomendadas

## 8.1 Aprendizaje y Referencia

### 8.1.1 Para seguir aprendiendo

| Recurso | Tipo | Ideal para |
|---------|------|------------|
| **Kaggle** | Datasets + cursos + competiciones | Practicar datos reales |
| **DataCamp / DataQuest** | Cursos interactivos | Aprender herramientas nuevas |
| **YouTube (Alex the Analyst, Chandoo)** | Tutoriales gratuitos | Soluciones rápidas |
| **Stack Overflow** | Foro técnico | Debugging |
| **Medium / Towards Data Science** | Artículos técnicos | Mantenerse actualizado |
| **GitHub** | Código abierto, proyectos ejemplo | Ver cómo trabajan otros |

### 8.1.2 Libros complementarios

Los libros de la colección *Data & Analytics Mastery* son el complemento perfecto para este. Si te falta base técnica:

- **Libro 1**: Fundamentos del Analista de Datos (SQL, Python, Excel)
- **Libro 2**: Python para Datos + BigQuery Cloud
- **Libro 3**: Business Intelligence con Power BI
- **Libro 4**: Ingeniería de Datos (dbt, Airflow, Docker)
- **Libro 5**: Big Data, ML e IA (PySpark, ML, LLMs)

## 8.2 Plataformas Freelance

Las plataformas freelance recomendadas incluyen Upwork como la más grande, donde conviene tener perfil completo y propuestas personalizadas; Fiverr para servicios empaquetados; Workana para el mercado hispanohablante; Toptal con tarifas altas pero proceso de selección exigente; Freelancer.com como alternativa a Upwork con menos comisión; PeoplePerHour popular en UK y Europa con proyectos frecuentes de datos; y LinkedIn ProFinder para clientes directos sin comisión.

## 8.3 Herramientas Freemium que Debes Conocer

| Herramienta | Función | Limitación gratuita |
|-------------|---------|---------------------|
| **Google Colab** | Jupyter en la nube | GPU limitada |
| **DuckDB** | SQL embebido, ultra rápido | Sin límite real |
| **Tableau Public** | Visualizaciones públicas | No puedes ocultar datos |
| **Power BI Desktop** | BI profesional | Gratuito |
| **Streamlit Community** | Dashboards Python en la nube | Apps públicas |
| **GitHub** | Control de versiones | Repos públicos gratis |
| **Superset (Apache)** | BI open source | Auto-hosteado |
| **Metabase** | BI open source, simple | Auto-hosteado |

## 8.4 Páginas para Encontrar Datos de Práctica

Para encontrar datos de práctica gratuitos, puedes usar Kaggle con miles de datasets reales de todos los temas, Google Dataset Search como buscador de datasets académicos y gubernamentales, Data.gov y Datos.gob.es para datos abiertos gubernamentales, Our World in Data para datos globales con visualizaciones, y FiveThirtyEight para datos limpios de periodismo de datos.

## 8.5 Comunidades para Freelancers de Datos

Las comunidades recomendadas incluyen Reddit r/datasets para compartir datasets y proyectos, Reddit r/dataisbeautiful para inspiración en visualizaciones, Discord de Data & Analytics como comunidad en español, LinkedIn Groups como Data Analytics Freelancers y Power BI Community, y Slack de DataTalksClub como comunidad grande y activa de datos.
