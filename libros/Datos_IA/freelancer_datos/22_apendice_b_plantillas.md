# Apéndice B: Plantillas y Recursos

## B.1 Plantilla de Propuesta Comercial

```markdown
**Propuesta de Análisis de Datos para [Empresa]**

**1. Resumen Ejecutivo**
- Qué vamos a hacer
- Por qué es importante
- Cuánto cuesta
- Cuándo estará listo

**2. Contexto y Objetivos**
- Situación actual
- Problema a resolver
- Objetivos del proyecto

**3. Alcance del Proyecto**
Incluye:
- [ ] Análisis de [fuente de datos]
- [ ] Dashboard con [X] métricas
- [ ] Informe ejecutivo
- [ ] Presentación de resultados

No incluye:
- [ ] Desarrollo de nuevas fuentes de datos
- [ ] Formación al equipo
- [ ] Mantenimiento continuo

**4. Metodología**
- Fase 1: Extracción y limpieza (X días)
- Fase 2: Análisis exploratorio (X días)
- Fase 3: Dashboards y visualización (X días)
- Fase 4: Documentación y entrega (X días)

**5. Entregables**
- Dashboard interactivo [enlace demo]
- Informe ejecutivo en PDF
- Documentación técnica

**6. Cronograma**
- Inicio: [fecha]
- Hito 1: [fecha]
- Entrega final: [fecha]

**7. Presupuesto**
- Inversión total: [X€]
- Formas de pago: 50% al inicio, 50% a la entrega
- Incluye: 2 rondas de revisión

**8. Siguientes Pasos**
- Aceptar propuesta
- Programar kickoff
- Firmar acuerdo
```

## B.2 Plantilla de Informe Ejecutivo

```markdown
**Informe Ejecutivo: [Título del Proyecto]**
**Para:** [Nombre del Cliente]
**Fecha:** [Fecha]
**Autor:** [Tu Nombre]

**Resumen**
- El objetivo de este análisis era [objetivo]
- Principales hallazgos:
  - Hallazgo 1
  - Hallazgo 2
  - Hallazgo 3

**KPIs Clave**
| KPI | Valor | vs. Período Anterior | Target | Estado |
|-----|-------|---------------------|--------|--------|
| Ejemplo | 250.000€ | +12% | 300.000€ | ⚠️ |

**Recomendaciones**
1. [Recomendación con impacto y esfuerzo estimado]
2. [Recomendación con impacto y esfuerzo estimado]
3. [Recomendación con impacto y esfuerzo estimado]

**Próximos Pasos**
- [ ] Validar hallazgos con el equipo
- [ ] Implementar recomendación 1
- [ ] Agendar siguiente revisión
```

## B.3 Plantilla de Contrato / Acuerdo

```markdown
**ACUERDO DE SERVICIOS DE ANÁLISIS DE DATOS**

Entre [Tu Nombre] ("Consultor") y [Cliente] ("Cliente")

**1. Servicios**
El Consultor proporcionará los siguientes servicios:
- [Descripción detallada]

**2. Compensación**
- Honorarios: [X€]
- Forma de pago: 50% inicio, 50% entrega
- Facturación: [tu método]

**3. Cronograma**
- Inicio: [fecha] | Entrega: [fecha]
- Se permite un retraso de hasta 5 días hábiles por imprevistos.

**4. Propiedad Intelectual**
Todo el código, dashboards y análisis pasan a ser propiedad del Cliente tras el pago completo.

**5. Confidencialidad**
El Consultor no compartirá datos del Cliente con terceros.

**6. Cancelación**
El Cliente puede cancelar en los primeros 7 días con reembolso del 100%. Después, no hay reembolso.

**7. Firmas**
_________________        _________________
Consultor                 Cliente
Fecha:                    Fecha:
```

## B.4 Plantilla de Briefing para Cliente

```markdown
**Briefing del Proyecto**

Datos del Cliente:
- Nombre de la empresa:
- Sector:
- Persona de contacto:
- Email / Teléfono:

Sobre el Proyecto:
1. ¿Cuál es el problema que quieres resolver?
2. ¿Qué preguntas quieres responder con los datos?
3. ¿Qué fuentes de datos están disponibles?
4. ¿Quién usará los resultados?
5. ¿Qué decisiones se tomarán con este análisis?
6. ¿Tienes algún deadline?
7. ¿Presupuesto estimado?
8. ¿Has trabajado antes con un analista de datos?

Datos Técnicos:
- ¿Dónde están los datos? (Excel, BD, API, data warehouse)
- ¿Tienes acceso directo o necesitas que alguien te lo proporcione?
- ¿Qué herramientas usas actualmente?
- ¿Hay alguien de TI que pueda apoyar?
