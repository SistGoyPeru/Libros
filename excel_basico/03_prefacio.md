# Prefacio

### ¿Por qué este libro?

Has visto el título. "Excel Básico: El Legado de las Fórmulas". Suena a manual, a curso técnico, a aburrimiento. Y sin embargo, este es un libro para aprender Excel de una manera completamente diferente.

Este libro nació de una convicción: **la mejor manera de aprender una herramienta es necesitarla de verdad.**

No te voy a engañar con ejercicios de "ingresa estos datos en la celda A1". Aquí vas a ayudar a Sofía Mendoza a salvar el taller de carpintería de su abuelo, a calcular costos, a organizar inventarios, a crear presupuestos, y a descubrir que detrás de cada número hay una historia que merece ser contada.

### ¿Cómo usar este libro?

Cada capítulo tiene tres partes:

1. **Narrativa** — La historia avanza. Sofía descubre el legado de su abuelo, enfrenta desafíos, comete errores, aprende y crece.
2. **Explicación técnica** — Los conceptos de Excel que Sofía usa para resolver sus problemas. Explicados con claridad, con ejemplos prácticos.
3. **Enigmas** — Ejercicios para que tú mismo practiques lo aprendido. Al final del libro encontrarás las soluciones.

### ¿Qué necesitas?

- Microsoft Excel 2019 o posterior (o Google Sheets como alternativa gratuita)
- Una computadora con sistema operativo Windows, Mac o Linux
- Curiosidad y ganas de aprender

### Un viaje, no un destino

Al terminar este libro, no solo sabrás cómo Sofía salvó el taller "Mendoza & Hijos". También habrás aprendido Excel desde la interfaz básica hasta las funciones más útiles, pasando por formato de celdas, gráficos, filtros y mucho más.

Y quizás, solo quizás, descubras que Excel no es solo una herramienta de oficina. Es un lienzo donde pintar tus ideas, un cuaderno donde organizar tus sueños, un legado que puedes dejar a quienes vienen después.
