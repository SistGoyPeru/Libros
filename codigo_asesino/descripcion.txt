En el Neo-Cusco del futuro, donde la tecnología inca y los algoritmos modernos chocan, la analista Wayra Condori debe descifrar un asesinato usando Python y pandas. Este libro combina una novela de misterio con una guía práctica de análisis de datos: listas, diccionarios, DataFrames, limpieza de datos y visualización. Ideal para quienes quieren aprender Python para datos mientras resuelven un crimen ancestral.
